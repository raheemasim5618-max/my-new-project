
# WhatsApp Screen Clone

This is a simple WhatsApp chat screen clone built using **Xcode**.  
It demonstrates the UI of a chat interface similar to WhatsApp.

---

## Features
- Chat bubbles (incoming and outgoing messages)
- Header with contact name and status
- Input field for typing messages
- Scrollable chat view

---

## How to Run
1. Open `WhatsAppClone.xcodeproj` in **Xcode**
2. Select a simulator or your device
3. Click **Run** (▶️ button) to launch the app
